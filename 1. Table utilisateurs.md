1\. Table utilisateurs

*Sert à gérer les connexions et les infos clients.*



id (INTEGER, Primary Key, Auto-increment)



nom (TEXT)



prenom (TEXT)



email (TEXT, Unique)



mot\_de\_passe (TEXT)



role (TEXT)



2\. Table tables\_restaurant

*Représente la configuration physique du restaurant.*



id (INTEGER, PK)



numero\_table (INTEGER)



capacite (INTEGER)



emplacement (TEXT)



3\. Table reservations

*C'est la table de liaison entre un utilisateur et une table à une date donnée.*



id (INTEGER, PK)



utilisateur\_id (INTEGER, Foreign Key vers utilisateurs.id)



table\_id (INTEGER, Foreign Key vers tables\_restaurant.id)



date\_heure (DATE)



nombre\_personnes (INTEGER)



statut (TEXT)
